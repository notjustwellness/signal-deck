The demand test.

Prove a stranger will pay — before you build the thing.

Every page before this one is a bet. This is the page that settles it. Most founders get the order wrong — build, then hope someone buys. We run it backwards: sell the promise first, and build only what a stranger has already paid for. You don't need a product to run this. You need an offer, a price, a ship date, and ten strangers.

The gate
Show the offer to ten target strangers. If at least three pay today, you build. If fewer than three, you stop and listen.
Money moved is the only vote that counts — not likes, not comments, not email sign-ups. A signup is curiosity. A charge is demand.
Three ways to run it · start in the middle
Tier 1
The fake door
A "Buy" button that logs intent before it charges. Cheapest to stand up, but the weakest signal — a click isn't a card.
~$20 · about an hour
Tier 2
The refundable deposit
A Stripe payment link takes a small, fully refundable deposit. Real money changes hands with almost no build. The honest, high-signal, low-cost test.
~$40 · half a day
Start here
Tier 3
The pre-sale
A full pre-order storefront that funds batch one. The strongest signal — only worth building once Tier 2 has already passed.
~$100 · a weekend
The five steps
  1. 1
    Write the offer
    One benefit headline [set at ZH], one honest price, one real ship month (~90 days out). You're selling the promise — the felt benefit and the date — not the science. Keep it to a paragraph.
  2. 2
    Build the page
    A single Carrd page, about thirty minutes. The offer, the price, one button. No navigation, no story, no "learn more." One decision.
  3. 3
    Wire the money
    A Stripe payment link for a refundable deposit. You do not need an LLC, a factory, or FDA clearance to take a refundable deposit. Thirty more minutes.
  4. 4
    Find ten strangers
    Ten people in the actual target — not ten friends. Friends buy to be kind; that's false validation, the most expensive kind. Post where the customer already is and ask them to pay, today.
  5. 5
    Run it, read the verdict
    Three or more pay → green light; build batch one. Fewer than three → stop, and email every non-buyer one question: "what stopped you?" The no is the most valuable thing you'll buy this month.
The ethics · non-negotiable

This is the honesty brand. Run the test the way the brand talks, or don't run it.

  • A real ship window, stated up front — never a date you can't hit.
  • One-click refunds, always, no questions.
  • If it's a no-go, refund everyone, immediately — the deposit was always theirs.
Money moved. Not likes.
