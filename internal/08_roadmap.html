Chapter 07

Known risks & soft spots

Named honestly, because an honesty brand can't hide its own gaps.

None is fatal; each has a stated answer or a watch.

  • No felt mirror (the retention risk). The product can't be felt, so adherence is the weak point. Answer: the dose-mirror narration + the named Rhodiola felt layer + the ritual — never a fabricated sensation (the full mechanic is doc 09). The founding cohort is the live test of whether it holds.
  • No RCT yet. We carry branded-ingredient pedigree, not a Signal-specific trial. Honest line: effect proof is borrowed + re-order data until a trial is funded by revenue.
  • Proving the pathway — the method is unsolved. The commitment is locked (Chapter 02): absorption proof is the differentiated edge of the moat. What's open is the specific method — a credible, affordable absorption test the bioavailability claim can lean on.
  • The velocity-vs-integrity fork. Veteran/tactical is chosen over CrossFit to avoid energy-drink capture; revisit only if growth stalls, and only if we can win that velocity without adopting that language.
  • COGS not yet validated. The cost stack is a bottom-up estimate with three real anchors; the validated quote is the first use of funds and the largest unknown in the model (doc 07).
Chapter 08

Parked — deliberately not now

Real items, deliberately held off the current path.
  • Serving-size test — "15-day first cycle → 30-day subscription" (Option B). The 30-day box stays the standard; this is a noted idea, not a model change, and the economics are deliberately not run here. The idea: ship the first order as a 15-day flagship trial (flagship only, same per-day price — lower commitment to try, no discount) that converts to the normal 30-day subscription, with the existing Day-14 dose-mirror email as the natural opt-in moment. The appeal: it forces the subscription decision sooner and surfaces an earlier retention signal in the riskiest window (the first weeks), without permanently doubling shipments — which a permanent 15-day pack would, at a real cost to margin and to the creatine companion. How we'd go about it, if ever pursued: a launch-window A/B test or a read off the founding cohort — a test, not a desk decision.
  • Manifesto reconciliation. The old manifesto is a different product generation (2-SKU, old formula, pre-redraw lines); it needs its own pass before it rejoins canon — not threaded in piecemeal.
  • Longevity / affluent scale. The large market, but last — unlocked only on re-order proof, never run as a Phase-1 seed.
  • Take-back / closed-loop packaging. Foregone for now; no promise we can't keep (recyclability honesty, doc 06).
  • Practitioner listing (Fullscript). Conversation open pre-launch; listing is a Phase-2+ decision after DTC proof.
  • The 5-year model refresh. Parked on the prior SKU numbering + benchmark COGS; rebuilt against the validated cost stack once the hero quote lands (doc 07).
